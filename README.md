# taxi-dispatch-api

## 실행 방법

### Mac 기준

1. rvm 설치
  - [Link](https://rvm.io/rvm/install)에서 설치법 확인 후 설치

2. Ruby 설치 및 설정

```bash
$ rvm list known
$ rvm install 2.7.1
$ rvm use 2.7.1
```

3. Project Setting

```bash
$ git clone https://github.com/nasang88/taxi-dispatch-api.git && cd taxi-dispatch-api/

$ bundle install

$ rails server
```
